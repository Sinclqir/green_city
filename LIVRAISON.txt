INSTRUCTIONS POUR LE DÉPLOIEMENT:
================================

1. Créer un compte GitHub et pousser le code
2. Créer un compte Vercel et connecter le repo
3. Créer un compte Codecov et connecter le repo
4. Configurer les secrets GitHub Actions:
   - VERCEL_TOKEN
   - VERCEL_ORG_ID  
   - VERCEL_PROJECT_ID
   - PROD_API_URL (URL Vercel du backend)

5. Créer une base de données MySQL en production (Aiven/AlwaysData)
6. Configurer les variables d'environnement Vercel avec les credentials de prod
